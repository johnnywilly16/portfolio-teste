# Portfolio Johnny

Portfolio pessoal desenvolvido com Next.js, Tailwind CSS, Framer Motion, Anime.js e GSAP.

## Características

- Animação de entrada com efeito de estrelas
- Design responsivo
- Efeitos de parallax
- Animações suaves
- Tema espacial

## Tecnologias Utilizadas

- Next.js 14
- Tailwind CSS
- Framer Motion
- Anime.js
- GSAP
- TypeScript

## Como Executar

1. Clone o repositório
```bash
git clone https://github.com/johnnywilly16/portfolio-teste.git
```

2. Instale as dependências
```bash
npm install
# ou
yarn install
```

3. Execute o servidor de desenvolvimento
```bash
npm run dev
# ou
yarn dev
```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador

## Estrutura do Projeto

- `/app` - Páginas e componentes principais
- `/components` - Componentes reutilizáveis
- `/public` - Arquivos estáticos
- `/styles` - Estilos globais
